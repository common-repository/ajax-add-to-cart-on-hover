== Description ==

Ajax add to cart on hover Plugin is used for adding variable products to cart using overlay on image when hovered over product image.

After installation and activation of the plugin it automatically shows up overlay on the variable product image with variations and you can add that product to cart by clicking on that product's variation.

This scenario will only happen in product-category and shop pages.


== Installation ==
From the admin panel, Go to your WordPress Admin -> Plugins -> Add New. Search for Ajax add to cart on hover. Install and Activate.

From directories,
1. Upload "Ajax add to cart on hover" to the "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress
